## 🔍 Overview

<a href="#airServer">AirServer</a> transforms a Mac into a wireless display receiver — accepting AirPlay streams from iPhone and iPad, Google Cast from Android devices, and Miracast from Windows devices, turning the Mac's display into a destination for screen mirroring from any mobile platform.

The <a href="#airServer">AirPlay 2 receiver capability</a> enables iPhone and iPad screen mirroring to the Mac without any cables — the same wireless mirroring that Apple TV provides, available through the Mac running AirServer on the local network. iPhones and iPads see the Mac running AirServer as an available AirPlay destination alongside any Apple TVs on the network, selecting it from the screen mirroring control center to mirror the iOS screen to the Mac display. The <a href="#airServer">multiple simultaneous streams</a> receives several device streams at once — displaying mirrored screens from multiple iPhones or iPads simultaneously in separate windows on the Mac, enabling side-by-side screen comparison, classroom device monitoring, and presentation contexts where multiple device displays are needed at once.

The <a href="#airServer">Google Cast support</a> receives screen mirrors and media casts from Android devices and Chrome browser — extending AirServer's platform coverage to the Android ecosystem alongside iOS. Android users cast their screen to the Mac running AirServer the same way iOS users AirPlay, making the Mac a universal screen receiver for mixed iOS and Android environments. The <a href="#airServer">screen recording capability</a> records any mirrored device screen to a video file — capturing device demos, app walkthroughs, gameplay, and any mirrored content as an H.264 video for sharing, documentation, or publishing.</p>

<a href="#airServer">Presentation display mode</a> provides a focused presentation view of a mirrored iOS device for conference room and classroom use — displaying the mirrored device screen prominently for audience visibility during demonstrations, educational sessions, and presentations. <a href="#airServer">Hardware-accelerated H.264 encoding</a> uses the Mac's GPU for video processing during screen recording — efficient, low-overhead recording that doesn't significantly impact the Mac's performance during simultaneous use.

<a href="#airServer">Miracast support</a> receives wireless display connections from Windows devices — providing screen mirroring from the Windows platform alongside iOS and Android. <a href="#airServer">AirServer Connect companion app</a> on iOS provides discovery and connection management for AirServer on Mac from the iOS device side — simplifying the connection process for educational and enterprise deployments where managed device connections are needed.</p>
